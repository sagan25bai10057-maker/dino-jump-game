# 🦖 Dino Jump Game
A simple browser dinosaur jumping game built with **HTML, CSS, and JavaScript**.
## 🎮 Features
- 🦖 Dinosaur jump mechanics
- 🪙 Coin collection
- ⭐ Star power-up (invincibility)
- 📈 Increasing difficulty
- 🏆 High score system
- 🖱 Mouse / keyboard / touch controls
## 🎮 Controls
| Control | Action |
|-------|------|
| Spacebar | Jump |
| Touch Screen | Jump |
| Mouse Cursor on Dinosaur | Jump |
## 📦 Technologies Used
- HTML
- CSS
- JavaScript
## 🚀 How to Run
1. Download the project
2. Open `index.html` in your browser
or run using Live Server in VS Code.
## 🌐 Play Online
You can host this game using **GitHub Pages**.
